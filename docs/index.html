Документация Smartford OS

Документация движка SmartFord OS: рендеринг .smapp, API (storage, network, prefs, UI) и выполнение цепочек событий.

description Структура

Файл читается функцией RenderApp, разбивается на секции и преобразуется в UI.

[Metadata] id=com.smartfordos.example.myapp title=Пример приложения permissions=["storage","network"] [Button1] type=button label=Нажми меня onClick=handleButtonClick buttonStyle=filled position=["middle", "bottom"] margin=["0", "2"] [Text1] type=text id=text1 content=Привет, SmartFord OS! position=["left", "top"] margin=["2", "0"] [Events] handleButtonClick=smartfordos.ui.update("text1","Clicked")

code Компоненты

Button

Кнопка создаётся через дублирование шаблона (filled_button, outlined_button и т.д.).

type=button label=Нажми меня buttonStyle=filled onClick=handleButtonClick position=["middle", "bottom"] margin=["0", "2"] id=btn1

Поддерживаемые стили:

outlined filled elevated tonal
Код:
type=button label=Нажми меня buttonStyle=tonal onClick=handleButtonClick position=["middle", "bottom"] margin=["0", "2"] id=btn1
Как выглядит:
Нажми меня

Text

type=text id=text1 content=Привет! size=18 font=Arial color=onSurface position=["left", "top"]

Поддерживаемые цвета:

primary onPrimary primaryContainer onPrimaryContainer secondary onSecondary secondaryContainer onSecondaryContainer surface onSurface onSurfaceVariant surfaceContainer surfaceDim outline shadow error onError

Также поддерживаются:
• HEX-формат: color="#FF5722"
• RGB-числовой: color=12345678
• По умолчанию: color="onSurface"

Поддерживаемые шрифты:
любые, которые установлены у пользователя (например, Arial, Times New Roman, Roboto и т.д.)

Позиционирование

position=["left", "top"] # горизонталь, вертикаль margin=["2", "0"] # горизонтальный, вертикальный отступ (в мм/10)

Горизонталь: left, middle, right
Вертикаль: top, middle, bottom, center

terminal События и цепочки

События выполняются через ExecuteEventChain и поддерживают цепочки команд через ->. (аналог коварного goto)

handleButtonClick=smartfordos.api.storage.create_dir(".test") -> onDirCreated onDirCreated=smartfordos.api.storage.write_file(".test/new.txt", "Hello!") -> handleRead handleRead=smartfordos.api.storage.file.read(".test/new.txt") -> updateUI updateUI=smartfordos.ui.update("text1", "{res}")

{res} - результат текущей команды
{prev_res} - результат предыдущей команды

api API

Storage API

smartfordos.api.storage.create_dir("folder") smartfordos.api.storage.write_file("file.txt","data") smartfordos.api.storage.file.read("file.txt")

Требует разрешения: storage

Network API

smartfordos.network.fetch("https://example.com") smartfordos.network.fetch.get("https://example.com") smartfordos.network.fetch.post("https://example.com", "body", "Content-Type: application/json") smartfordos.network.fetch.delete("https://example.com")

Требует разрешения: network

Prefs API (хранилище настроек)

smartfordos.prefs.set("key","value","String") smartfordos.prefs.get("key") smartfordos.prefs.delete("key") smartfordos.prefs.type("key") smartfordos.prefs.list()

Типы: String, Boolean, Number

UI API

smartfordos.ui.update("text1","New text")

Обновляет текст компонента по id или AlternativeText

System API

smartfordos.system.wait("3000") smartfordos.system.call("hyperlink","https://google.com") smartfordos.system.call("macros","ModuleName.MacroName")

wait - задержка в миллисекундах
hyperlink - открытие ссылки (с проверкой безопасности)
macros - запуск макроса VBA

security Разрешения

  • storage - доступ к файловой системе
  • network - HTTP запросы

shield Безопасность

В SmartFord OS реализованы механизмы безопасности (в сетевых запросах) для защиты пользователей от вредоносного контента и обеспечения безопасного взаимодействия с приложениями. Например, SmartFord OS блокирует:
  • .ua домены
  • NSFW (adult) контент
  • Телеграмм каналы (кроме разрешенных: https://t.me/SmartfordOS, https://t.me/AndreyGreece1606, https://t.me/nestoros5, https://t.me/vanko_corporation", https://t.me/TSD_TML_OFFICIAL, https://t.me/Chel_v_trusax)
  • Редиректы (могут содержать все что угодно, включая NSFW, и др.)
  • HTTP запросы (разрешен только HTTPS)
